# Griya Hydrofarm Website

Website modern dan responsive untuk Griya Hydrofarm - usaha sayuran hidroponik di Lampung Selatan.

## Fitur

- ✅ Design modern dengan tema hijau
- ✅ Fully responsive (desktop & mobile)
- ✅ Integrasi WhatsApp untuk pemesanan
- ✅ Form pemesanan otomatis ke WhatsApp
- ✅ UI/UX interaktif
- ✅ Rating Google 5.0
- ✅ Informasi kontak lengkap
- ✅ Floating WhatsApp button
- ✅ Optimized untuk SEO

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide Icons

## Cara Menjalankan

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka browser di [http://localhost:3000](http://localhost:3000)

## Deploy ke Vercel

1. Push code ke GitHub repository
2. Import project di [Vercel](https://vercel.com)
3. Deploy otomatis akan berjalan
4. Website siap digunakan!

### Deploy via Vercel CLI

```bash
npm install -g vercel
vercel login
vercel
```

## Struktur Website

- **Hero Section**: Tampilan utama dengan CTA

- **Products Section**: Produk selada hijau
